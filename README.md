# Hyprland-dotflies
Yet another Hyprland dotfiles. Made for a dark-themed, minimalist look with a homemade expressive pastel color palette.

## Red Rose Palette
![Red Rose Color Palette](https://github.com/Sim0nePL/hyprland-dotflies/blob/main/Red%20Rose.png)

## Config
These dotfiles provide configs for:
- [x] hyprland
- [x] kitty
- [x] wofi ( including some wofi menus )
- [x] mako
- [x] cava
- [ ] waybar
- [ ] gtk/qt themes

The default apps are:
- Terminal: **Kitty**
- Browser: **Zen Browser**
- File Manager: **Nemo**
- Apps Launcher: **Wofi**
- Clipboard Manager: **Clipse**
- Notification Manager: **Mako**

## Hyprland Shortcuts Overview
- **SUPER + R** -> Apps Launcher <sub>( Run Menu )</sub>
- **SUPER + F** -> File Manager
- **SUPER + B** -> Browser
- **SUPER + V** -> Clipse Menu

- **SUPER + Arrow** -> Move window focus
- **SUPER + Space** -> Switch window floating
- **SUPER + BACKSPACE** -> Close App

There are keybinds integrations for hyprshot, hyprpicker, hyprzoom etc. u can check in keybinds section in hyprland.conf
